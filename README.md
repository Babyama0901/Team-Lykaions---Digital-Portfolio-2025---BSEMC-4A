# Team Lykaions - Digital Portfolio 2025

This project is a digital portfolio for the Seminar and Field Study (BSEMC 4A). It is built using React and Vite, featuring a flipbook component for presenting the portfolio content.

## Technologies Used

- **React**: UI library
- **Vite**: Build tool and development server
- **TailwindCSS**: Utility-first CSS framework
- **react-pageflip**: Library for the flipbook effect

## Getting Started

To run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Babyama0901/Team-Lykaions---Digital-Portfolio-2025---BSEMC-4A.git
    cd Team-Lykaions---Digital-Portfolio-2025---BSEMC-4A
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    ```

4.  Open your browser and navigate to the local server URL (usually `http://localhost:5173`).

## Project Structure

- `src/components`: Contains React components (Book, Library, etc.)
- `src/Seminars and Field Study - Portfolio`: Contains portfolio images
- `public`: Static assets
