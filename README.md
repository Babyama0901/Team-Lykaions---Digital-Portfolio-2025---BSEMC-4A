# Team Lykaions - Digital Portfolio 2025

This project is a digital portfolio for the Seminar and Field Study (BSEMC 4A) class. It features an interactive library and a realistic 3D flipbook component to present student portfolios.

**[üåê Live Demo](https://babyama0901.github.io/Team-Lykaions---Digital-Portfolio-2025---BSEMC-4A/)**

## Features
- **Interactive Library**: Browse through student portfolios in a visual grid layout.
- **3D Flipbook**: Realistic page-turning effects for an immersive reading experience.
- **Mobile Responsive**: Fully optimized for mobile devices, ensuring accessibility on all screen sizes.

## Technologies Used
- **React**: UI library
- **Vite**: Build tool and development server
- **TailwindCSS**: Utility-first CSS framework
- **react-pageflip**: Library for the flipbook effect

## Getting Started

To run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Babyama0901/Team-Lykaions---Digital-Portfolio-2025---BSEMC-4A.git
    cd Team-Lykaions---Digital-Portfolio-2025---BSEMC-4A
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  Open your browser and navigate to the local server URL (usually `http://localhost:5173`).

## Project Structure

- `src/components`: React components (Book, Library, Footer, etc.)
- `src/Seminars and Field Study - Portfolio`: Portfolio image assets
- `src/Logo`: Project logos and branding
- `public`: Static assets
